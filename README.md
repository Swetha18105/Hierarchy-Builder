# 🌳 Hierarchy Builder

A full-stack web application that processes hierarchical relationships and generates structured output from given input data.

---

## 🚀 Live Demo

- 🔗 Frontend: https://mellifluous-cannoli-8a6ea9.netlify.app/
- 🔗 Backend: https://hierarchy-builder-qjz3.onrender.com/

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Deployment:**  
  - Frontend → Netlify  
  - Backend → Render  

---

## 📌 Features

- Accepts input in:
  - JSON format  
  - Comma-separated values  
- Processes hierarchical relationships (e.g., `A->B`)
- Generates structured API response  
- Handles invalid inputs gracefully  
- Clean and responsive UI  

---

## 📥 Input Format

### JSON Input
```json
{
  "data": ["A->B", "B->C"]
